📊 Google Sheet Integration

Connect a Google Sheet as your portfolio database. The sheet is the source of truth for which coins are tracked and their holdings.

One-time setup via Google Apps Script (5 min)

1Open your Google Sheet → click Extensions → Apps Script.
2Delete any existing code and paste the script shown below. Click Save (💾).
3Click Deploy → New deployment. Type: Web app. Execute as: Me. Who has access: Anyone. Click Deploy.
4Copy the Web app URL (starts with https://script.google.com/macros/s/…/exec) and paste it below.

Paste this code in Apps Script (step 2)

function doGet() {
  var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
  var data  = sheet.getDataRange().getValues();
  if (data.length < 2) return out([]);
  var headers = data[0].map(function(h) {
    return String(h).toLowerCase().trim().replace(/\s+/g,'_');
  });
  var rows = data.slice(1).map(function(row) {
    var obj = {};
    headers.forEach(function(h,i){ obj[h] = row[i]; });
    return obj;
  }).filter(function(r){ return r.coin_id; });
  return out(rows);
}
function out(data) {
  return ContentService.createTextOutput(JSON.stringify(data))
    .setMimeType(ContentService.MimeType.JSON);
}
