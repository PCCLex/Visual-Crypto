📊 Google Sheet Integration

Connect a Google Sheet as your portfolio database. The sheet is the source of truth for which coins are tracked and their holdings.

How to set up your sheet

1Go to sheets.google.com and create a new spreadsheet. Name it e.g. Visual Crypto Portfolio.
2Add the column headers in row 1 exactly as shown below (copy-paste friendly).
3Fill in your coins from row 2. Use the CoinGecko ID (e.g. bitcoin, ethereum, solana).
4Go to File → Share → Publish to web. Choose Sheet1 + Comma-separated values (.csv). Click Publish.
5Copy the published URL (or just the regular share URL) and paste it below.

coin_id name symbol holdings total_cost invested_eur (optional)
